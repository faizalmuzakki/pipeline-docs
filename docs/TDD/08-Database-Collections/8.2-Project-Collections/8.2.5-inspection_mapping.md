# 8.2.5 inspection mapping

Column mapping configuration for Excel import (maps Excel columns to database fields)

| Field | Data Type | Description |
|-------|-----------|-------------|
| _id | String | Primary key (ObjectId as string) |
| id_item | String | Item reference (ObjectId as string) |
| id_spec | String | Specification reference (ObjectId as string) |
| test_type | String | Test type (sample: "product_analysis", "heat_analysis", "tensile", "charpy") |
| data | Object | Column mapping configuration object with dynamic fields based on test_type |
| data.[field] | String | Dynamic mapping fields (value: "default" or custom column mapping) |
| created_at | Date | Creation timestamp (ISODate) |
| created_by | String | Creator user email |
| updated_at | Date | Last update timestamp (ISODate) |
| updated_by | String | Last updater user email |

## Dynamic Data Fields by Test Type

The `data` object contains dynamic fields that vary based on the `test_type`:

**Chemical Analysis (product_analysis, heat_analysis):**
- `data.c`, `data.si`, `data.mn`, `data.p`, `data.s` - Basic elements (Carbon, Silicon, Manganese, Phosphorus, Sulfur)
- `data.v`, `data.nb`, `data.ti`, `data.cu`, `data.ni` - Alloying elements (Vanadium, Niobium, Titanium, Copper, Nickel)
- `data.cr`, `data.mo`, `data.al`, `data.n`, `data.b` - Additional elements (Chromium, Molybdenum, Aluminum, Nitrogen, Boron)
- `data.pcm`, `data.ce` - Calculated values (PCM, Carbon Equivalent)

**Mechanical Tests (tensile, charpy, hardness):**
- Fields vary based on specific test requirements (e.g., yield_strength, tensile_strength, elongation, impact_energy, hardness_value)

**Note**: Each field maps an Excel column to a database field. Value "default" indicates standard column position, or can be a custom column identifier for non-standard Excel layouts.
