# Project Key Terms Endpoints

| API Name | API Route | HTTP Method |
|----------|-----------|-------------|
| Get key terms for datatables | /api/v3/project/key_terms/datatables | GET |

| Delete key term | /api/v3/project/key_terms/{id} | DELETE |
